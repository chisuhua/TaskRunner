## Phase 3.3a — Event Timing Precision (Week 1)

### Problem (from Phase 1.7 review)

当前 `cu_event.cpp` 有一个**隐藏的精度 bug**（Phase 2 PoC 遗留）：

```cpp
struct EventTable {
  std::unordered_map<CUevent, std::chrono::steady_clock::time_point> created;
};

// 创建时
table.created[*phEvent] = std::chrono::steady_clock::now();

// 记录时（cuEventRecord）
it->second = std::chrono::steady_clock::now();  // ⚠️ 覆盖创建时间！

// 计算时（cuEventElapsedTime）
auto diff_us = std::chrono::duration_cast<std::chrono::microseconds>(
    end_it->second - start_it->second);  // ⚠️ 如果两个事件都没 Record，diff = 创建时间之差
```

**Bug 分析**：
- `created` 字段同时承担"创建时间"和"记录时间"两个角色
- `cuEventRecord` 会覆盖创建时间（语义混乱）
- `cuEventElapsedTime` 在两个事件都未 Record 时返回错误的数据
- 没有 CUDA 时钟语义对齐（虽然 CudaStub 永远是同步执行的）

### Goal

修复 bug，统一时间源模型，添加以下 P1 Phase 3.3a 交付物：

| API | Phase 1.7 状态 | Phase 3.3a 目标 |
|-----|----------------|-----------------|
| `cuEventCreate` | REAL_IMPL (buggy) | **修复** + 支持 `CU_EVENT_BLOCKING_SYNC` / `CU_EVENT_DISABLE_TIMING` flags 语义 |
| `cuEventCreateWithFlags` | STUB (delegates) | **REAL_IMPL** (proper flag handling) |
| `cuEventRecord` | REAL_IMPL (buggy) | **修复** + 写 `recorded` 而非覆盖 `created` |
| `cuEventSynchronize` | REAL_IMPL (no-op) | **保持 no-op** + 添加 optional 真实 fence wait |
| `cuEventQuery` | REAL_IMPL (no-op) | **保持 no-op** + 添加 optional completion check |
| `cuEventElapsedTime` | REAL_IMPL (buggy) | **修复** + 正确处理"未 Record"情况 |
| `cuEventDestroy` | REAL_IMPL (OK) | **保持** |

**新增** (Phase 3.3a 范围):
- `cuEventRecordWithFlags` (CUDA 12.x)：flags 参数支持
- 内部状态机：`created` / `recorded` / `destroyed` 三态机

#### B.2 cuEventCreate 修复（30 min）

```cpp
extern "C" CUresult cuEventCreate(CUevent* phEvent, unsigned int Flags) {
  if (!phEvent) return CUDA_ERROR_INVALID_VALUE;
  // 拒绝未知 flag（CUDA spec: reserved bits must be 0）
  constexpr unsigned int kValidFlags = CU_EVENT_DEFAULT
                                    | CU_EVENT_BLOCKING_SYNC
                                    | CU_EVENT_DISABLE_TIMING
                                    | CU_EVENT_INTERPROCESS;
  if (Flags & ~kValidFlags) return CUDA_ERROR_INVALID_VALUE;

  auto& table = async_task::umd::shim::g_events;
  std::lock_guard<std::mutex> lock(table.mu);
  CUevent id = reinterpret_cast<CUevent>(
      static_cast<std::uintptr_t>(table.next_id.fetch_add(1)));
  table.events[id] = EventRecord{
      .flags = Flags,
      .created_at = std::chrono::steady_clock::now(),
      .recorded_at = std::nullopt,
      .is_destroyed = false,
  };
  *phEvent = id;
  return CUDA_SUCCESS;
}
```

#### B.3 cuEventRecord 修复（30 min）

```cpp
extern "C" CUresult cuEventRecord(CUevent hEvent, CUstream hStream) {
  if (!hEvent) return CUDA_ERROR_INVALID_VALUE;
  auto& table = async_task::umd::shim::g_events;
  std::lock_guard<std::mutex> lock(table.mu);
  auto it = table.events.find(hEvent);
  if (it == table.events.end()) return CUDA_ERROR_INVALID_HANDLE;
  if (it->second.is_destroyed) return CUDA_ERROR_INVALID_HANDLE;

  // 写入 recorded_at 而非 created_at
  it->second.recorded_at = std::chrono::steady_clock::now();
  (void)hStream;  // Phase 3.3a: 保持 no-op (stream 不影响时间)
  return CUDA_SUCCESS;
}
```

#### B.4 cuEventElapsedTime 修复（1 h）

```cpp
extern "C" CUresult cuEventElapsedTime(float* pMilliseconds, CUevent hStart, CUevent hEnd) {
  if (!pMilliseconds) return CUDA_ERROR_INVALID_VALUE;
  auto& table = async_task::umd::shim::g_events;
  std::lock_guard<std::mutex> lock(table.mu);

  auto start_it = table.events.find(hStart);
  auto end_it = table.events.find(hEnd);
  if (start_it == table.events.end() || end_it == table.events.end())
    return CUDA_ERROR_INVALID_HANDLE;
  if (start_it->second.is_destroyed || end_it->second.is_destroyed)
    return CUDA_ERROR_INVALID_HANDLE;

  // 严格语义：如果任一事件未 Record，必须先 Record
  if (!start_it->second.recorded_at.has_value() ||
      !end_it->second.recorded_at.has_value()) {
    // Phase 2 行为：fallback 到 created_at（向后兼容）
    // Phase 3.3a 决策：返回 NOT_PERMITTED（更严格）
    return CUDA_ERROR_NOT_PERMITTED;
  }

  auto diff_us = std::chrono::duration_cast<std::chrono::microseconds>(
      *end_it->second.recorded_at - *start_it->second.recorded_at);
  *pMilliseconds = static_cast<float>(diff_us.count()) / 1000.0f;
  if (*pMilliseconds < 0.0f) *pMilliseconds = 0.0f;
  return CUDA_SUCCESS;
}
```

**关键决策**: 是否保留 Phase 2 的"未 Record fallback 到 created_at"行为？
- **保守** (Q4 review 推荐): 保持向后兼容，fallback 仍然有效
- **严格** (符合 CUDA spec 12.x): 返回 `CUDA_ERROR_NOT_PERMITTED`

→ 决策：**保守** (本子计划内)，后续 Phase 3.x 可加严。

#### C.2 cuArray 实现 (3-4 h)

```cpp
extern "C" CUresult cuArrayCreate(CUarray* pHandle,
                                   const CUDA_ARRAY_DESCRIPTOR* pAllocateDescriptor) {
  if (!pHandle || !pAllocateDescriptor) return CUDA_ERROR_INVALID_VALUE;
  // 校验 descriptor
  if (pAllocateDescriptor->numChannels < 1 || pAllocateDescriptor->numChannels > 4)
    return CUDA_ERROR_INVALID_VALUE;

  auto desc = std::make_unique<ArrayDescriptor>();
  desc->format = pAllocateDescriptor->format;
  desc->num_channels = pAllocateDescriptor->numChannels;
  desc->width = pAllocateDescriptor->width;
  desc->height = pAllocateDescriptor->height;
  if (pAllocateDescriptor->format == CU_AD_FORMAT_3DC4 ||
      pAllocateDescriptor->format == CU_AD_FORMAT_3DC8) {
    desc->depth = pAllocateDescriptor->depth;
  } else {
    desc->depth = 1;  // 1D/2D
  }
  // 计算 total size + pitch
  auto bytes_per_channel = (desc->format == CU_AD_FORMAT_FLOAT) ? 4 : 2;
  desc->pitch = desc->width * desc->num_channels * bytes_per_channel;
  desc->total_size_bytes = desc->pitch * desc->height * desc->depth;
  desc->backing.resize(desc->total_size_bytes, 0);

  auto& table = async_task::umd::shim::g_arrays;
  std::lock_guard<std::mutex> lock(table.mu);
  CUarray id = reinterpret_cast<CUarray>(
      static_cast<std::uintptr_t>(table.next_id.fetch_add(1)));
  table.arrays[id] = std::move(desc);
  *pHandle = id;
  return CUDA_SUCCESS;
}

extern "C" CUresult cuArrayDestroy(CUarray hArray) {
  if (!hArray) return CUDA_ERROR_INVALID_VALUE;
  auto& table = async_task::umd::shim::g_arrays;
  std::lock_guard<std::mutex> lock(table.mu);
  auto it = table.arrays.find(hArray);
  if (it == table.arrays.end()) return CUDA_ERROR_INVALID_HANDLE;
  if (it->second->is_destroyed) return CUDA_ERROR_INVALID_HANDLE;
  it->second->is_destroyed = true;
  table.arrays.erase(it);
  return CUDA_SUCCESS;
}

extern "C" CUresult cuArrayGetDescriptor(CUDA_ARRAY_DESCRIPTOR* pArrayDescriptor,
                                          CUarray hArray) {
  if (!pArrayDescriptor || !hArray) return CUDA_ERROR_INVALID_VALUE;
  auto& table = async_task::umd::shim::g_arrays;
  std::lock_guard<std::mutex> lock(table.mu);
  auto it = table.arrays.find(hArray);
  if (it == table.arrays.end() || it->second->is_destroyed)
    return CUDA_ERROR_INVALID_HANDLE;
  pArrayDescriptor->format = it->second->format;
  pArrayDescriptor->numChannels = it->second->num_channels;
  pArrayDescriptor->width = it->second->width;
  pArrayDescriptor->height = it->second->height;
  if (it->second->depth > 1) {
    pArrayDescriptor->depth = it->second->depth;
  }
  return CUDA_SUCCESS;
}
```
